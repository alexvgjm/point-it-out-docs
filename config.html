Configuring global options

The config function receives a SystemOptions object and will update only the global options presents in it. In the current version there is only one option.

import { config } from 'pointitout' 

config({
    updateOnResize: true 
})

SystemOptions

Name
Type/s
Description
updateOnResize Default: true
boolean
If true, the update function will be called on every window resize, updating all pointers.
