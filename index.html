Getting started

Point it out is a tiny library to mark/point out DOM elements using another absolutely positioned elements or procedurally generated SVGs.

The most common use case is to easily create tutorial pointers to point out interface elements that you want to describe or that the user should interact with.

Installation

Install pointitout from npm:

npm install pointitout

or inject the single minified .js file as a <script> from: https://unpkg.com/pointitout@latest/pointitout.js

Basic usage

import { create } from 'pointitout' 

create('rect', { target: '#target-element-id' })

See more options for the create function.
