Point it out

A tiny (~1.3 Kb gzipped), zero-dependencies JavaScript library to point out DOM elements using another absolutely positioned elements or procedurally generated SVGs.

Getting started

Installation

Install pointitout from npm:

npm install pointitout

Basic usage

create('rect', {
    target: '#example',
    strokeColor: "#5896d5",
    strokeWidth: 8,
    round: '20%',
})
#example

See more options for the create function.
